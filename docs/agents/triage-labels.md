# Triage 라벨

| 표준 역할 | GitHub 라벨 | 의미 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 유지관리자 검토 필요 |
| `needs-info` | `needs-info` | 제보자의 추가 정보 대기 |
| `ready-for-agent` | `ready-for-agent` | 에이전트가 추가 질의 없이 구현 가능 |
| `ready-for-human` | `ready-for-human` | 사람의 구현 또는 판단 필요 |
| `wontfix` | `wontfix` | 진행하지 않음 |

스킬이 역할 이름을 언급하면 이 표의 GitHub 라벨을 적용한다.
